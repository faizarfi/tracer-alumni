# Sistem Pendataan dan Pelacakan Alumni (Tracer Study)
## UIN Raden Mas Said Surakarta

Sistem berbasis web ini dikembangkan menggunakan **Framework Laravel** untuk mempermudah proses pendataan, pelacakan karir, dan manajemen data alumni di lingkungan UIN Raden Mas Said Surakarta.
